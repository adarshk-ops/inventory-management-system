# Inventory Management System

A web-based inventory system built using Java, JSP, Servlets, and MySQL.

## Features
- Add Product
- View Products
- Update Product
- Delete Product
- Low Stock Alert
- Search Functionality

## Technologies Used
- Java (Servlets, JSP)
- JDBC
- MySQL
- Apache Tomcat

## How to Run
1. Import project into Eclipse
2. Configure Tomcat Server
3. Run the project
4. Open: http://localhost:8080/InventoryWebApp/viewProducts.jsp
